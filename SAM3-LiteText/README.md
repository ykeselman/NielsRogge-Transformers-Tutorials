# SAM-3 Lite-Text notebook

This folder contains a notebook to compare inference of Meta's SAM-3 with SAM-3 Lite-Text, which is largely the same except it reduces the number of parameters of the text encoder by 88%.

* docs: https://huggingface.co/docs/transformers/main/en/model_doc/sam3_lite_text
* checkpoints: https://huggingface.co/models?other=sam3_lite_text
